# React Login Page App

This repository contains a single-page React application that renders a polished login experience without connecting to any backend. It focuses on UI/UX, client-side validation, and responsiveness.

## Development

```bash
cd frontend
npm install
npm start
```

The app opens at `http://localhost:3000` and updates live as you edit the source code.

## Tests & Build

```bash
npm test
npm run build
```

The production-ready build lives in `frontend/build` after running the build command.
